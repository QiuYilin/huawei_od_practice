CSDN上大量的相关专栏应该比本项目更好，附带的OJ也比本项目简单的CTest更精确。

本项目主要只是本人对题目的重新归类和小量补充。

分类规则：

    不同目录必然有交叉

    模拟题也可以归类为查找，排序，核心思想是找到合适的数据结构，所以也可以归类到对应的数据结构中，特指数据结构和算法都不突出的情况

    如果模拟题的规律性比较明显，且不需要动态表辅助，就纳入贪心

练习顺序：

    ①模拟：快速适应ACM模式，习惯IO输入输出，功利角度节省审题时间，题量较大，适合周期性训练

    ②贪心：因人而异，模拟的规律不容易想到就纳入贪心，功利角度上来说得分收益最大，题量较大，适合周期性训练
	
	③图论：最小生成树仅有一题，题量小分值大辨识度比较高
