题目描述
有N个快递站点用字符串标识，某些站点之间有道路连接。 每个站点有一些包裹要运输，每个站点间的包裹不重复，路上有检查站会导致部分货物无法通行，计算哪些货物无法正常投递？

输入描述
第一行输入M N，M个包裹N个道路信息.
0<=M,N<=100，
检查站禁止通行的包裹如果有多个以空格分开
输出描述
输出不能送达的包裹，如：package2 package4，
如果所有包裹都可以送达则输出：none，
输出结果按照升序排列。
用例1
输入
4 2
package1 A C
package2 A C
package3 B C
package4 A C
A B package1
A C package2
输出
package2
